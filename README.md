# MCBE-SERVER-CHAT
Clone the repo and run
`npm install`
`npm run compile`
Create a .env file with a contents of 
```
TOKEN=
ACCOUNTUSENAME=
GUILDID=
CHANNELID=
IP=
PORT=
```
and fill with the appropriate info

finally use `node dist/index.js` to run.
